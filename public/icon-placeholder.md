# PWA Icons Required

This app needs the following icon files in the `public` directory:

- `icon-192x192.png` - 192x192 pixels
- `icon-512x512.png` - 512x512 pixels
- `favicon.ico` - Standard favicon

You can generate these icons using:
- Online tools like https://realfavicongenerator.net/
- Design tools like Figma, Canva, or Photoshop
- Or use a simple image generator

The icons should be square PNG images with transparent backgrounds (for maskable icons).
